• Welcome to Qalculate! Click Next to begin the tutorial

  • Start by clicking on the cell and typing 2+2 then Enter

  • Pressing Enter runs the cell,
    Create a new math cell by hovering over the current cell and clicking +Math

  • Now try a square root. Type sqrt5 in the cell

  • Now click on the cell and press Esc (or Tab )
    Notice how the cell is no longer outlined in blue You are now in command mode
    Press a to insert a cell above
    Press b to insert a cell below Press m to convert a cell to markdown

  • Now type and run L_0Tab=3 ft (in a math cell)
    Insert a new cell below and type and run L0^2
    Now edit the second cell to L0^2 to m^2
    Notice how the to command converts between units

  • Edit the first cell to be L_0 = 12 ft
    Re-run all the cells by pressing Alt + r

  • Create a markdown cell ( Esc, b, m or +Markdown )
    Type # Space Heading 1 Enter ## Space Heading 2 Enter Hello, World!

  • You have now learned the basics of the notebook!
    For advanced features, try selecting text and pressing

    • ctrl + i
    • ctrl + b
    • ctrl + u
    See what happens when you type * Text or ---
    Try pasting images into a markdown cell
